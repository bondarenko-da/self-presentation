# Дмитрий Бондаренко - Senior Developer & DWH Architect

Профессиональный landing page для презентации опыта и компетенций в области Data Engineering и разработки корпоративных хранилищ данных.

## 🚀 Технологии

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React

## ✨ Особенности

- 🎨 Современный светлый дизайн
- 📱 Полностью адаптивный интерфейс
- ⚡ Быстрая загрузка и оптимизация
- 🎬 Плавные анимации и переходы
- 🔍 SEO-оптимизация

## 📋 Секции

- **Hero** - Главная секция с фото и кратким описанием
- **О себе** - Профессиональный профиль
- **Навыки** - Технический стек с визуализацией уровня
- **Опыт работы** - Timeline с историей работы
- **Проекты** - Ключевые проекты для Enterprise-компаний
- **Образование** - Образование и сертификаты
- **Контакты** - Контактная информация

## 🛠️ Локальный запуск

```bash
# Установка зависимостей
bun install

# Запуск в режиме разработки
bun run dev

# Сборка для продакшена
bun run build
```
